# Связный список (Linked List)

Связный список — это базовая динамическая структура данных, состоящая из узлов. Каждый узел содержит данные и ссылку (или ссылки) на следующий и/или предыдущий узел в последовательности. В отличие от массива, элементы в связном списке не хранятся в непрерывной области памяти.

## Основные характеристики

*   **Динамический размер:** Память выделяется по мере добавления новых элементов.
*   **Последовательный доступ:** Для доступа к элементу необходимо пройти по ссылкам от начала списка (доступ за O(n)). Произвольный доступ отсутствует.
*   **Эффективность вставки/удаления:** При наличии указателя на нужную позицию, вставка и удаление элементов выполняются за O(1) (в массиве — O(n) из-за сдвига).

## Односвязный список (Singly Linked List)

Каждый узел хранит данные и ссылку только на **следующий** узел. Навигация возможна только в одном направлении (от головы к хвосту).

graph LR
    Head[Head] --> Node1(Данные | next);
    Node1 --> Node2(Данные | next);
    Node2 --> Node3(Данные | next);
    Node3 --> Null(null);
